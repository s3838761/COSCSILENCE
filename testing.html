Testing

The first interface (Registration interface)


The components
(The application logo, Google account signing in button and different email accounts button).
Test results
In general, all the components of the first interface have been tested and proven to be effective and successful. First, the application's logo and buttons for registration appeared successfully in the area specified for it in the design. Second, the registration was tested with a Google account which then transferred the user to a sub-interface and successfully allowed entering of the user email and password. Third, the registration was tested with another email account (Hotmail), which then transferred the user to a sub-interface and successfully allowed entering of the user email and password.

The second interface (Home interface)


The components
(Welcome message with username, user photo, rewards progression bar).
Test results
In general, all the components of the second interface have been tested and proven to be effective and successful. First, the welcome message with the username and the user's picture appeared successfully in the area specified for it in the design. Second, the reflection of task completion on the rewards progression bar has been tested and proven to be effective.

The third interface (information interface)


The components
(Child's photo, Child's first name, Child's last name, Child's email, Child's age, Guardian's first name, Guardian's last name and Guardian's email).
Test results
All components of the third interface have been tested and proven effective and successful, as the child's picture, the child's first name, the child's family name, the child's email, the child's email, the child's age, the first guardian's name, the guardian's family name and the guardian's email appeared successfully in the area specified for it in the design.

The fourth interface (Tasks interface)


The components
(The task wheel, the child's first name, the number of tasks chosen, the tasks to be completed).
Test results
In general, all the components of the fourth interface have been tested and proven to be effective and successful. First, the task wheel was tested on the child's completion of some tasks, the reversal was done successfully, and the task wheel showed the tasks that the child had successfully completed and their percentage. Second, the reflection of the number of selected tasks was tested on the child's completion of some tasks, the reflection was done successfully, and it showed the number of tasks chosen by the child successfully. Third, the ability of the child to choose the tasks to be performed was tested, and the application succeeded in proving its effectiveness.

The fifth Interface (Rewards Interface)


The components
(Interface title, username, available rewards options for the child, Redeem button).
Test results
In general, all components of the fifth interface have been tested and proven to be effective and successful. First, the interface address and username have successfully appeared in the area specified for it in the design. All the reward options available to the child have been tested by selecting and then redeeming from the redeem button and all have been successfully proven to be effective.
