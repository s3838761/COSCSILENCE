Tools

To complete this assignment our group used various tools/software. The various tools we used are as follow:

  • Microsoft Teams
    • Microsoft teams was as the main hub of group communication. All meetings were held in Microsoft teams and even the interview was conducted on Microsoft teams. Furthermore, links or files that needed to be shared amongst the whole team was shared here. These links include but not limited to the GitHub repository link and the Google doc’s link.
  • GitHub
    • GitHub was mainly used to share the website amongst the team members and have it so that all members can work on it at any given time. As well as this the repository was used to host GitHub pages to view the website as a whole.
  • Visual Studio Code
    • Visual Studio Code was used code the entire website with all the HTML and CSS.
  • Google Docs
    • The team used Google Doc’s as a hub to collect all written content while also being able to edit and add content simultaneously.
  • Adobe Photoshop
    • Adobe Photoshop was briefly used to edit images that would be used on the website.

Using all these tools we as a team have investigated 4 IT technologies. These IT technologies we decided to investigate were Robots, Autonomous Vehicles, Clouds Services and Servers and Cybersecurity. As well as this the team was able to conduct an interview on an IT profession which gave us a valuable insight on life as someone working in the IT field. Following on from this we also analyzed industry data on IT jobs and compared it with all our ideal jobs. Furthermore, we were able to come up with a solid project idea which would be developed in Assignment 3.
